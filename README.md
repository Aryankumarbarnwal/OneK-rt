# 🛒 OneK@rt

## 🚀 About the Project

Hi! 👋
I built **OneK@rt** as part of my learning journey in full-stack web development. While learning the MERN stack, I wanted to create something practical and close to real-world applications, so I decided to build an e-commerce platform similar to Flipkart/Amazon.

This project took me around **1 month** to complete, and I built it **individually**, which helped me understand how a complete application works from frontend to backend and admin control.

---

## 🧠 What I Learned

While building this project, I got hands-on experience with:

* Full-stack development (Frontend + Backend + Database)
* Authentication and Authorization using JWT
* Payment integration using Razorpay
* Image upload and management with Cloudinary
* Google Authentication using Firebase
* State management and routing in React
* Real-world project structure (Admin + User + Backend)

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT, Firebase (Google Login)
* **Payment Integration:** Razorpay
* **Media Storage:** Cloudinary

---

## 🏗️ Project Structure

The project is divided into three main parts:

* **Frontend** → User interface (React app)
* **Backend** → Server & APIs (Node + Express)
* **Admin Panel** → Platform management system

---

## 🌐 Deployment

The **frontend and backend are deployed on Render**.

⚠️ Since the project is hosted on a free tier, it may respond a bit slowly sometimes (especially during the first request due to server cold start).

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/OneK@rt.git
```

2. Navigate to each folder and install dependencies

```bash
cd frontend
npm install

cd ../backend
npm install

cd ../admin
npm install
```

3. Setup environment variables (.env)

* MongoDB connection string
* JWT secret
* Razorpay keys
* Cloudinary credentials
* Firebase config

4. Run the project

```bash
npm run dev
```
